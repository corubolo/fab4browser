Notices: 

  • This software is still under development, there are known limitations.
  • The original documents are never modified by Fab4 Browser annotations, and the notes/changes are visible only using Fab4.
  • Public annotations will be seen by anybody using Fab4 open the document. 
  • You can save to private notes so your notes will be private, in a local database. See also the user manual.
